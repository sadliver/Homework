2.differentiation tag

Name tag สิ่งที่เหมือนกัน สิ่งที่ต่างกัน
hr & br เป็นตัวแบ่ง element hr ใช้สร้างเส้น
br ใช้สำหรับขึ้นบรรทัดใหม่
ul & ol ใช้สร้างข้อความรูปแบบ list ul เป็นการเรียงรูปแบบสัญลักษณ์
ol เป็นการเรียงรูปแบบตัวเลข
select & checkbox สร้างตัวเลือก select เลือกอย่างใดอย่างหนึ่ง
checkbox เลือกได้มากกว่า1ข้อ
radio & button เป็นเครื่องหมายสามารกดได้ radio เป็นปุ่มตัวเลือกวงกลม
button เป็นปุ่มสี่เหลี่ยม
div & span กำหนดส่วน div ใช้จัดกลุ่มองค์ประกอบ block level
span ใช้เพื่อจัดกลุ่มองค์ประกอบแบบ inline
script &style แท๊กการกำหนดรูปแบบ script ใช้กำหนดในรูปแบบสคริปต์
style ใช้กำหนดในรูปแบบข้อมูลสไตล์

3. input type

Input type หลักการใช้
"text" กำหนดรูปแบบที่เป็นตัวหนังสือ
"button" กำหนดปุ่มกด
"checkbox" เป็นรูปแบบปุ่มกดแบบตัวเลือก สามารถเลือกได้มากกว่าหนึ่ง
"color" การกำหนดรูปแบบของสี
"date" การกำหนดรูปแบบของวันที่
"datetime-local" การกำหนดเขตข้อมูลวันที่และเวลาโดยไม่มีเขตเวลา
"email" การกำหนดรูปแบบอีเมล
"file" การกำหนดรูปแบบไฟล์
"image" การทำหนดรูปแบบรูปภาพ
"month" การทำหนดรูปแบบเดือน
"number" การทำหนดรูปแบบตัวเลข
"password" ใช้รับค่าอักษรเป็นรหัสผ่าน ซึ่งจะถูกปิดบังไว้เป็นเครื่องหมาย *
"radio" เป็นรูปแบบปุ่มกดแบบตัวเลือกอย่างใดอย่างหนึ่ง
"range" การทำหนดรูปแบบตัวเลขแบบจำกัด
"reset" ใช้กำหนดปุ่มรีเซ็ตที่จะรีเซ็ตค่าแบบฟอร์มทั้งหมดเป็นค่าเริ่มต้น
"search" การหนดรูปแบบค้นหา
"submit" ใช้สำหรับส่งข้อมูลฟอร์มไปยัง form-handler
"tel" การกำหนดรูปแบบตัวเลขโทรศัพท์
"time" การกำหนดรูปแบบเวลา
"url" การกำหนดรูปแบบลิงค์
"week" การกำหนดรูปแบบสัปดาห์
